Use Postman to test routes! Postman example Download Postman

List of routes for managing whatsapp:

All whatsapp connection wheels have a pattern of Headers of user. The headers must be parameterized as :

{
    "Content-Type" : "application/json",
    "user" : "user",
    "user_pass" : "user"
}
